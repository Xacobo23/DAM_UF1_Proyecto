# Descripción del Proyecto

En esta sección, se proporcionará una visión general del proyecto en cuestión, destacando sus objetivos, alcance y contexto general.

El objetivo del proyecto es crear una aplicación para visualizar el tiempo y otros datos metereologicos de un concello concreto.

El alcance del proyecto se corresponde con las siguientes funcionalidades principales:

- Comprobar el estado actual del concello.
- Comprobar las distintas predicciones del concello.
- Ver el estado de las posibles alertas activas.
- Permitir de una forma sencilla la interaccion del usuario con la interfaz




## Justificación del Proyecto

En este apartado, se argumenta la razón y necesidad detrás del proyecto. Se explican los problemas o desafíos que se abordarán y por qué es relevante llevar a cabo el proyecto.

La razón para lleva a cabo el proyecto es crear una aplicación para ver el tiempo sencilla y accesible, que se ajuste a las demandas del usuario.

El desafío principal de este proyecto sería que la aplicación tenga la mayor accesibilidad posible y permita un uso fácil e intuitivo, con valores detallados y explicados.

La relevancia del proyecto va de la mano con el objetivo principal
